**美食魔物世界观**

## 世界观概述

在魔王封印的那日，世界的边陲小镇出现了一座高耸入云的“塔“，塔内的魔物具有可食用的部位。

例如头顶苹果派的苹果派红鸟，金黄的派皮上铺满了苹果散发着果香；腿部是香喷喷炸鸡的鸡腿菇；尾巴是极佳油的油脂猪。

食用这些部位甚至可以带来强大的魔力提升。

神秘的高塔得到高度的重视，各地的冒险小队皆向往成为登上高塔顶层的第一人。

## 魔物设定概览
| 魔物名称 | 简介 |
| :------ | :--- |
| 苹果派红鸟 | 所到之处会散发着苹果派的甜香，将头顶的苹果派视为帽子，如果酥脆的派皮不小心破损，会难过地吃掉帽子，等待新的帽子生长。和它同种族的还有香蕉派黄鸟，蓝莓派蓝鸟。 |
| 鸡腿菇 | 本体是普通的菇类，腿部是香喷喷的炸鸡，据说跑得越快的鸡腿菇腿部的炸鸡会变得越柴。推荐捕捉懒惰的鸡腿菇食用。 |
| 油脂猪 | 尾巴储藏着丰厚的油脂，偶尔会滴落在地面。沿着油脂寻找踪迹一定能发现油脂猪的群落。 |
| 香草麻雀 | 会叽叽喳喳吵醒冒险者的魔物，尾羽是上好的香草。因为体型娇小，行动敏捷，即使能力弱小也难以捕捉。 |
| 热锅龟 | 背部沸腾着香料丰富的锅底，辣度视魔物的颜色而定，越深的热锅龟背部的火锅越辣。推荐自带食材。 |
